# OmicClaw

OmicClaw is the conversational AI layer of OmicVerse. It lets you run
analyses through natural language via messaging platforms (Telegram, Feishu,
iMessage, QQ) and through the Model Context Protocol (MCP) for Claude Code
and other AI agents.
