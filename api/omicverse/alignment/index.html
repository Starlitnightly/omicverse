omicverse.alignment¶

source package omicverse.alignment

Alignment analysis utilities.

This module provides comprehensive tools for fastq data processing and alignment including

  • Alignment with kb-python
  • RNA velocity analysis with kb-python
  • SRA download / conversion / QC / alignment / counting wrappers

Functions

  • ref — Build kallisto index and transcript-to-gene mapping via kb ref. Returns a dict with metadata and common output paths.

  • count — Generate count matrix from single-cell FASTQ files via kb count. Returns a dict with metadata and commonly generated output files if present.

  • parallel_fastq_dump — Download SRA data in parallel using parallel-fastq-dump.

  • prefetch — Prefetch SRA accessions with validation.

  • fqdump — Convert SRA accessions to FASTQ.

  • fastp — Run fastp QC.

  • STAR — Run STAR alignment.

  • featureCount — Run featureCounts on BAM files.

  • bulk_rnaseq_pipeline — Run a complete bulk RNA-seq pipeline from SRA accessions or local FASTQs.

source parallel_fastq_dump(sra_id: str, threads: int = 1, outdir: str = '.', tmpdir: Optional[str] = None, min_spot_id: int = 1, max_spot_id: Optional[int] = None, split_files: bool = False, gzip: bool = False, **kwargs) → Dict[str, Union[str, int]]

Download SRA data in parallel using parallel-fastq-dump.

This function wraps the parallel-fastq-dump tool to download sequencing data from NCBI SRA (Sequence Read Archive) in parallel for faster downloads.

Parameters

  • sra_id : str — SRA accession ID (e.g., 'SRR2244401').

  • threads : int — Number of threads to use for parallel download. Default: 1.

  • outdir : str — Output directory for downloaded FASTQ files. Default: '.'.

  • tmpdir : Optional[str] — Temporary directory for intermediate files. Default: None.

  • min_spot_id : int — Minimum spot ID to download. Default: 1.

  • max_spot_id : Optional[int] — Maximum spot ID to download. Default: None (all spots).

  • split_files : bool — Split paired-end reads into separate files. Default: False.

  • gzip : bool — Compress output files with gzip. Default: False.

  • **kwargs — Additional arguments to pass to parallel-fastq-dump.

Returns

  • result — Dictionary containing download metadata including sra_id, threads, outdir, and output file paths.

Examples 

import omicverse as ov
# Download SRA data with 4 threads and split files
result = ov.alignment.parallel_fastq_dump(
    sra_id='SRR2244401',
    threads=4,
    outdir='fastq_output/',
    split_files=True,
    gzip=True
)
# Download with spot range limit
result = ov.alignment.parallel_fastq_dump(
    sra_id='SRR2244401',
    threads=8,
    outdir='fastq_output/',
    min_spot_id=1,
    max_spot_id=100000,
    split_files=True,
    gzip=True
)

source bulk_rnaseq_pipeline(sra_ids: Optional[Union[str, Sequence[str]]] = None, samples: Optional[Union[Tuple[str, str, Optional[str]], Sequence[Tuple[str, str, Optional[str]]]]] = None, genome_dir: str = 'star_index', gtf: str = 'genes.gtf', output_dir: str = 'pipeline_output', genome_fasta_files: Optional[List[str]] = None, threads: int = 8, memory: str = '50G', jobs: Optional[int] = None, skip_download: bool = False, skip_qc: bool = False, gzip_fastq: bool = True, gene_mapping: bool = True, auto_install: bool = True, overwrite: bool = False) → pd.DataFrame

Run a complete bulk RNA-seq pipeline from SRA accessions or local FASTQs.

The pipeline chains: prefetch -> fqdump -> fastp -> STAR -> featureCount.

Parameters

  • sra_ids : str or list of str, optional — SRA accession IDs to download. Required unless samples is provided.

  • samples : tuple or list of tuples, optional — Pre-existing FASTQ sample tuples (name, fq1, fq2_or_None). When provided, the download step is skipped automatically.

  • genome_dir : str — Path to (or for) the STAR genome index directory.

  • gtf : str — Path to the GTF annotation file.

  • output_dir : str — Root output directory. Sub-directories are created per step.

  • genome_fasta_files : list of str, optional — Genome FASTA file(s) for auto-building the STAR index.

  • threads : int — Threads per tool invocation.

  • memory : str — Memory limit for STAR BAM sorting (e.g. '50G').

  • jobs : int, optional — Number of concurrent jobs. None auto-detects.

  • skip_download : bool — Skip the prefetch + fqdump steps (requires samples).

  • skip_qc : bool — Skip the fastp QC step.

  • gzip_fastq : bool — Compress FASTQ output from fqdump.

  • gene_mapping : bool — Map gene_id to gene_name in featureCounts output.

  • auto_install : bool — Auto-install missing CLI tools via conda/mamba.

  • overwrite : bool — Force re-run even when outputs already exist.

Returns

  • pandas.DataFrame — Merged gene-level count matrix (genes x samples).

Raises

  • ValueError

  • RuntimeError
